## 돈/은행/환전
## 1,000원 + 1,000원 = 2,000원
## 2,000원과 2,000원은 같다.(equals)
## 돈은 음수일 수 없다.
## 5$ + 5$ = 10$
## 5$ - 6$ = 오류
## 5.25$ + 5.25$ = 10.50$ (소숫점 이하 2자리)
## 은행을 통해서 환전할 수 있다.
## 통화는 달러화와 원화만이 존재하고, 환율은 1달러 <-> 1,000원
## 1,000원 -환전-> 1$
## 5.25$ -> 5,250원
## 달러 -> 원화: 5원 이상 -> 10원으로 반올림
## 원화 -> 달러: $0.005 이상 -> $0.01 반올림
